WORKFLOW PROCESS

How It Works

A step-by-step guide to the digital borrowing process.

Class Creation

Teacher creates a class for their respective lab advisory.

Registration

Teacher registers students via class list.

Student Login

Student logs in using their Student ID.

Email Verification

Student verifies WMSU email using OTP.

Account Secured

Student creates password and completes setup.

Activity Setup

Faculty create lab activities.

Group Accountability

Leader assigns borrowed items to members.

Digital Reservation

Students pre-order equipment.

QR Code Release

Each request generates a QR code.

Secure Return

Technician scans items upon return.

Activity Setup

Faculty create lab activities and select required apparatus.

Digital Reservation

Students pre-order equipment.

QR Code Release

Each request generates a QR code.

Secure Return

Technician scans items upon return.

Shop & Checkout

Browse inventory, select items, and checkout digitally.

Approval Scan

Faculty approve the digital reservation.

Return & Log

Students return items and log the transaction.

Inspect & Log

Students inspect items for damage and log the findings.

Settlement Option

Faculty choose the appropriate settlement option for damaged items.

Proof & Receipt

Students submit proof of damage and receive a receipt.

Record Cleared

Damage records are cleared upon successful settlement.
